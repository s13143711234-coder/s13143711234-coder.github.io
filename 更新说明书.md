## C. ⭐ 最关键的计算规则（改错会出大问题）

### 排产（看板4 → SCHEDULE）：必须"各子类加总"
Excel 单元格里常是多类相加的文本，**必须把所有数字加起来**：
- 电池：`57.5铁锂+18三元` → 75.5（铁锂+三元+钴酸锂全加）
- 正极：`5500三元+500钴酸锂+9000铁锂` → 15000（全加）
- **隔膜：`湿法120000干法4500` → 124500（湿法+干法都要算！这里最容易漏）**

### 单位换算（Excel 原始值 → 网页显示）
- 电池：GWh，**直接用**
- 正极 / 负极 / 电解液：吨 **÷10000 = 万吨**
- 隔膜：万平 **÷10000 = 亿平**

### 其他重要约定
- 电池排产取 **"鑫椤"口径**那一行（排除名字里带 `-内部`/`-专家`/`-公司口径` 的行）。
- **8月内部排产数据禁止上看板**（Excel 里若有 8 月预测也不要放）。7月已获用户授权可放。
- **商用车比乘用车晚约1个月发布**：看板1常出现"乘用车已更新、商用车待补"，补最新月即可。
- 材料价格中 磷酸铁锂/负极/电解液/六氟/LiFSI/VC/加工费按"万元/吨"显示；隔膜按"元/㎡"。
- 图表默认显示**近一年**；归一化对比基准 = **2025-06-30 = 100**。
- 看板4"分企业月度排产"有**密码保护，密码：`htdx`**。

## D. 读 Excel 的技术要点

- Excel 里的日期是**序列号**（如 46173），换算：`[DateTime]::FromOADate(46173).ToString('yyyy-MM-dd')`。
- 用 `.Value2` 读原始数值（不是 `.Text`）。
- 从文本单元格提取并求和：正则 `\d+(\.\d+)?` 匹配所有数字后相加。
- 材料价格 `SMMDAY` 日度数据为控制文件体积，只保留 **2022-01 起**（早期留 null 断点）。

## E. PowerShell / 编码坑（否则中文乱码或页面空白）

- 含中文的 `.ps1` 脚本必须存成 **UTF-8 带 BOM**，否则子进程用 GBK 解码报错。
- 读写 HTML 用 `[IO.File]::ReadAllText/WriteAllText(path, text, [Text.Encoding]::UTF8)`，**不要**用 `Get-Content/Set-Content`（默认 UTF-16，破坏中文）。
- PowerShell 里 `"$yr年"` 会被当成变量名 `$yr年`（中文是合法标识符字符）→ 用 `"${yr}年"`。
- 不要用 `$c.Replace('</script>', ...)` 追加脚本——它会替换**所有** `</script>`（包括外部库标签）。要用唯一锚点定位。
- 改完务必检查 `<script>` 段里 `{` 和 `}` 数量相等（不等 = 语法坏 = 页面空白）。
